Project Title: Super Boogsh - Personal Portfolio Website
Author: Ejay Dimayuga

Project Purpose
The "Super Boogsh" website is a personal portfolio designed to showcase my skills, education, hobbies, and professional achievements. It serves as a digital resume and a platform for visitors to learn more about me, my expertise, and how to get in touch. The website is visually engaging, interactive, and responsive, ensuring a seamless experience across all devices.

Features

Hero Section:
A welcoming introduction with a typewriter effect displaying the website's title and a call-to-action button.

About Me Section:
A brief introduction about me, accompanied by a profile picture and links to my social media profiles.

Skills Section:
A showcase of my technical and professional skills using interactive flip cards and circular progress bars with animations.

Education Section:
A timeline of my educational background, presented with flip cards for an engaging visual experience.

Hobbies Section:
A carousel highlighting my favorite games, with thumbnails for quick navigation and hover effects for interactivity.

Contact Section:
A contact form with validation to ensure all fields are filled correctly before submission.
SweetAlert popups for success and error messages.

Background Music:
Background music with a control button to play or pause the audio.

Loading Animation:
A spinner animation displayed while the website loads, ensuring a smooth user experience.

Smooth Scrolling:
Smooth navigation between sections for a polished and professional feel.

External Resources and Libraries

Bootstrap 5:
Used for responsive design, grid layout, and components like the carousel and navbar.

AOS (Animate On Scroll):
Used for scroll-based animations to enhance the visual appeal of the website.

SweetAlert2:
Used for displaying custom alerts for form submission success or validation errors.

Google Fonts:
Font: "Press Start 2P" for a retro gaming aesthetic.

Animate.css:
Used for additional animations in SweetAlert popups.

Bootstrap Icons:
Used for icons in the skills section, social media links, and music control.

How to Run or View the Website

Clone or Download the Repository:
Clone the repository or download the project files to your local machine.

Open the Website:
Open the index.html file in any modern web browser (e.g., Chrome, Firefox, Edge).

Ensure Internet Connectivity:
The website uses external libraries (e.g., Bootstrap, AOS, SweetAlert2) via CDN links. Ensure you have an active internet connection for these resources to load.

Optional:
To modify or customize the website, open the project in a code editor like Visual Studio Code.

Additional Notes
The website is fully responsive and optimized for both desktop and mobile devices.
Background music is enabled by default but can be paused using the music control button.
The website includes smooth scrolling and animations for a modern and engaging user experience.
